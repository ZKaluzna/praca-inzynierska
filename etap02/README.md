# Etap 02: Pierwszy etap parametryzacji

Folder zawiera narzędzia do optymalizacji parametrów fizjologii i środowiska.

### Instrukcja uruchomienia
1. Otwórz plik `parametryzacja1.py`.
2. Zmień wartość zmiennej `TARGET_POPULATION` na pożądaną nazwę (np. `Arima1`, `Arima5`, `Aripo1` lub `Aripo6`).
3. Uruchom skrypt komendą:
   ```bash
   python parametryzacja1.py
