# Etap 02: Pierwszy etap parametryzacji

Folder zawiera pliki niezbędne do optymalizacji parametrów fizjologii gupików, fizjologii drapieżników oraz środowiska.

### Instrukcja uruchomienia
1. Otwórz plik `parametryzacja1.py`.
2. Zmień wartość zmiennej `TARGET_POPULATION` na nazwę pożądanej populacji (`Arima1`, `Arima5`, `Aripo1` lub `Aripo6`).
3. Uruchom skrypt za pomocą komendy:
   ```bash
   python parametryzacja1.py
