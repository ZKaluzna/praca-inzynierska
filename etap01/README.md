# Etap 01: Konstrukcja modelu bazowego

Ten folder zawiera pliki niezbędne do uruchomienia modelu w wersji podstawowej. Reprezentuje on interakcje między gupikami a środowiskiem, **bez uwzględnienia** preferencji samic oraz selekcji drapieżników.

### Instrukcja uruchomienia
Aby otworzyć interfejs modelu, wykonaj w terminalu komendę:
`python -m solara run app.py`

*Uwaga: Na tym etapie parametry modelu mają wartości poglądowe, służące weryfikacji poprawności działania samej aplikacji.*
