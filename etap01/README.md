# Etap 01: Podstawowa wersja modelu

Ten folder zawiera pliki niezbędne do uruchomienia modelu w wersji podstawowej, symulującej interakcje gupików ze sobą nawzajem oraz ze środowiskiem, **bez uwzględnienia** preferencji samic oraz selekcji drapieżników.

### Instrukcja uruchomienia
Aby otworzyć interfejs modelu, wykonaj w terminalu komendę:
`python -m solara run app.py`

*Uwaga: Na tym etapie parametry modelu mają wartości poglądowe, służące weryfikacji poprawności działania samej aplikacji.*
